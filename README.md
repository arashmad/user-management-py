## Installation

```bash
# Install Poetry
pipx install poetry
# Download the source code
git clone https://github.com/arashmad/user-management-py.git
# Install dependencies
cd fastAPI-poetry-boilerplate
poetry install
# Activate virtual environment
poetry shell
# Test the code and installation
make lint && make test
```
